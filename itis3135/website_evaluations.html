CRAP Website Evaluations

Yale School of Art

  • Contrast: This wouldn't actually have been so bad if it wasn't for the background. Because depending on where you are on the page, white text blends into the window of the background. There is just too harsh of a difference where it ends up with you struggling to find all the text visible.
  • Repetition: Depending on what their goal was, they accomplished creating a basic website using linear gradients. However, the only thing that is different is their calendar which doesn't have a gradient.
  • Alignment: In landscape, the alignment is poor. Everything is aligned to the right with no content anywhere else. In portriat however, everything is centered. It only works well in portrait.
  • Proximity: In general the proximity is okay. Only thing I'd change is moving their address to the footer.

Landbook

  • Contrast: Nothing really to comment on. Only nit pick is that the content is differentiated. The images are fine, but there is text underneath that should probably be in a container with the image and with a separate background color to make it easier to distinguish.
  • Repetition: Everything is lined up perfectly with each other. Each container is one column, and advertisements are two columns.
  • Alignment: Everything is perfectly aligned into columns and rows of equal spacing between each one.
  • Proximity: Each container is separated enough that you can clearly tell where it starts and ends.
